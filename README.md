`Works on macOS and Linux.`

## Install

```
$ npm install --global whatiz-cli
```
__`OR`__
```
$ sudo npm install --global whatiz-cli
```


## Usage
```
 Usage   : whatiz [package-name]

 Example : whatiz browserify
```
